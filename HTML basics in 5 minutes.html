Basics of HTML in 5 minutes

General HTML

What is HTML?

HTML (Hyper Text Markup Language) is the standard language for developing internet pages. Most likely, every webpage you've ever visited has been made in HTML. All common browsers (Google Chrome, Microsoft Edge/Internet Explorer, Firefox, Opera) all clearly present HTML. In this document, you will learn HTML5.

Why should I learn it?

As mentioned earlier, HTML is the standard language for coding websites. Even if you plan a career not into web development, learning this will help you out a lot.

Getting Started

Before you go coding, you need materials. This includes a text editor (which edits text, surprise), a file (to store the code somewhere), and basic computer skills (copy and paste, basics of how a computer works, software requirements). Scroll down to see your requirements.
Text Editors:

  • Sublime Text
  • Notepad++
  • Visual Studio Code

Okay, now that you know that, now its time for the fun:

In your computer, open a new text file and rename it something simple like "firstproject.html". It doesn't matter what the name of the file is, but the file must end with .html.
Open it with your text editor.

Now type the following:

<!DOCTYPE>

<head>

<title> My first title! </title>

</head>

<body>

<p> This is my first HTML sentence! Woo hoo! </p>

</body>

Now, save it manually or the recommended way (ctrl + s), and open it up on your browser!

Congrats! You made a sentence.

Let's analyse it. Hopefully, the first thing you noticed is this:
<body>
This is called a tag. It contains a < symbol, a few letters, and a > symbol.
What does this do? It tells whatever is inside of it (whatever is in the < and >) to function in a specific way. Another thing you should notice is that it is surrounded by TWO tags. This will limit what is inside or outside the tag!
You will also notice that the ending tag has a "/". Why? This will tell the code that it will end there.
The tag is the most important function in HTML.

Now, delete everything and just type in the following and refresh (or reopen) your file.
This is my second HTML sentence! Yee haw!
Why do does this sentence appear even though it has no tag to tell it to appear?

That is because HTML is a very forgiving language.
This is the main reason why HTML is a great language for beginners- one tiny mistake won't break the whole picture.
You will learn about this in depth in later chapters.

Experiment with these and many other tags. Some tags, you'll notice, won't end with an ending tag, like <br> or <hr>. That is because they don't accept text "inside" of them.
